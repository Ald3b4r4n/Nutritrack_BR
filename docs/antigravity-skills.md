# Antigravity Awesome Skills: NutriTrack BR

Este documento registra as skills do Antigravity Awesome Skills selecionadas para o projeto NutriTrack BR, justificando seu uso e garantindo a conformidade com a Constitution v1.0.0.

## Skills Selecionadas

### 1. `devcontainer-setup`
- **Justificativa**: Garante um ambiente de desenvolvimento isolado e reproduzível para todos os contribuidores, alinhado ao princípio de Governança (XVII).
- **Uso**: Configuração inicial do repositório para suportar VS Code Dev Containers com Flutter pre-instalado.

### 2. `context7-auto-research`
- **Justificativa**: Auxilia na pesquisa profunda de APIs do Flutter e Dart (Riverpod, Drift) para garantir implementações seguindo as melhores práticas atuais, alinhado ao princípio de Performance (IX).
- **Uso**: Acionada durante a fase de implementação de banco de dados e gerenciamento de estado para resolver ambiguidades técnicas.

### 3. `speckit-*` (Bundle)
- **Justificativa**: Essencial para o Spec-Driven Development (Princípio I) obrigatório na Constitution.
- **Uso**: Utilizada em todas as fases de especificação, planejamento e geração de tarefas.

## Conformidade com a Constitution

- **Privacidade**: Nenhuma skill de telemetria ou processamento de dados sensíveis fora do ambiente local será utilizada.
- **TDD**: Skills de testes serão priorizadas para automatizar a execução do ciclo Red-Green-Refactor.
- **Segurança**: Apenas skills do catálogo oficial [Antigravity Awesome Skills](https://github.com/sickn33/antigravity-awesome-skills) são permitidas.
